地址 0x01 
自定义功能码 0x41  读一次AD转换值
            0x42   读多次AD值

功能码 0x41    读一次AD转换值

请求      
功能码          一个字节    0x41
ADG通道选择     一个字节    0x00~0x03

响应
功能码         一个字节    0x41   
寄存器值       四/三个字节   0x00000000~    AD转换通道+AD数据 或者 ADG通道选择号+AD数据   或24位AD数据  取决于是否使能奇偶校验

错误
差错码         一个字节        0xc1
异常码         一个字节        0x01    不支持功能
               一个字节        0x02   ADG设置模式异常
               一个字节        0x03   从设备操作超时 
               一个字节        0x04   AD数据受损 
               
功能码 0x42    读多次AD转换值

请求      
功能码          一个字节    0x42
ADG通道选择     一个字节    0x00~0x03
AD转换次数      一个字节    0x01~0xff

响应
功能码         一个字节    0x42   
寄存器值       四/三个字节   0x00000000~    AD转换通道+AD数据   或24位AD数据  取决于是否使能回传状态寄存

错误
差错码         一个字节        0xc2
异常码         一个字节        0x01    不支持功能
               一个字节        0x02   ADG设置模式异常
               一个字节        0x03   从设备操作超时 
               一个字节        0x04   转换次数错误
